# Executive Sales & Profit Dashboard (Power BI)

## Overview
This project presents an executive-level Power BI dashboard designed to analyze sales performance, profitability, growth trends, and the impact of discounts.

## Key Features
- Executive Summary KPIs (Sales, Profit, GM%, YoY Growth)
- Sales & Profit Trend Analysis
- Product & Country Performance
- Discount Impact Analysis
- Business Insights & Recommendations

## Tools Used
- Power BI
- Microsoft Excel
- PowerPoint

## Files Included
- Power BI Dashboard (.pbix)
- Executive Presentation (PPT)
- Dashboard Export (PDF)

## Business Insights
- Identified the most profitable country
- Highlighted products with high sales but low profit
- Analyzed the impact of discounts on profitability
- Provided actionable business recommendations
